About Me

Hi, my name is Pedro Luna and I'm an IoT enthusiast. I've been working around this topic for ~6 years and have adopted diverse roles on several projects. From the very beginning, I have intended to learn the whole path to enable an IoT application, that, has lead me to learn a lot of things, from configure/program motes, to deploy, operate and maintain LoRa cloud services, or code end-user applications.

Throught my way on mastering the Internet of things I have discovered two important things:

  • Always be open to new skills, ideas, and technologies.
  • Have as much fun as you can in the process.

Hence, I've decided to get some fun by building this server with the following specs:

  • Server instance running on AWS.
  • The instance is running on Linux.
  • Http service, Email notifications and TTN connector coded in Go.
  • Go-routines installed as OS service.
  • Some content is delivered using Go and an HTML template, some other (like this text) is plain HTML since I don't have plans to change it frequently.

So, feel free to take a tour! If there is something that calls your attention and you like to know more about, you can send me a message using the form below or simply writing an email to clutso@hotmail.com.
